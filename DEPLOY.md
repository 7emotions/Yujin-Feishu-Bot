## Prerequisites
- OS: Ubuntu 22.04+
- Required: git, Python 3.11+, tmux
- All commands below assume a fresh Ubuntu server

## 1. Install System Dependencies
```bash
sudo apt update && sudo apt install -y git python3.11 python3-pip tmux
```

## 2. Install Node.js 18 via nvm
(lark-cli is a Node.js package, required at runtime)
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install 18
nvm use 18
nvm alias default 18
```

## 3. Install and Authenticate lark-cli
```bash
LARK_CLI_NO_PROXY=1 npm install -g @larksuite/cli
LARK_CLI_NO_PROXY=1 lark-cli auth login
```
Note: `lark-cli auth login` is interactive — it will open a browser URL for authorization.

## 4. Clone the Repository
```bash
git clone https://github.com/7emotions/feishu-reimbursement-bot.git
cd feishu-reimbursement-bot
```

## 5. Configure Environment Variables
```bash
cp .env.example .env
nano .env
```
Fill in the following variables:
```
APP_ID=cli_xxxx
APP_SECRET=xxxx
OPENAI_API_KEY=sk-xxxx
APPROVAL_CODE=xxxx
BOT_USER_ID=ou_xxxx
APPROVER_OPEN_ID=ou_xxxx
CONFIRM_KEYWORDS=确认,confirm,yes,ok
CANCEL_KEYWORDS=取消,cancel,no
TIMEOUT_SECONDS=300
```

## 6. Install Python Dependencies
```bash
pip install -r requirements.txt
```
Note: `requirements.txt` is generated by the bot setup.

## 7. Start the Bot
```bash
tmux new-session -s feishu-bot
```
Inside the tmux session:
```bash
./start_bot.sh
```
Detach with `Ctrl+B D`. To re-attach later:
```bash
tmux attach -t feishu-bot
```

## 8. Update the Bot
```bash
git pull
./restart_bot.sh
```
